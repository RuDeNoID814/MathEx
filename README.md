# MathEx

PWA-карточки по математическому анализу — определения, теоремы, формулы для подготовки к микросессии.

## Запуск

Статический сайт, никаких зависимостей. Открыть `index.html` локально или поднять любым http-сервером:

```bash
python3 -m http.server 8000
# открыть http://localhost:8000
```

## Структура

```
index.html          точка входа PWA
manifest.json       PWA-манифест
assets/             иконки
src/
  css/style.css
  js/
    app.js          UI, поиск, фильтры
    data.js         база карточек (вопросы → определения)
    graphs.js       рендер графиков
```

## Стек

HTML + CSS + ES5 JS, без сборщика. MathJax 3 для формул.
