# MathEx

PWA-шпаргалка для письменного экзамена по математическому анализу.

Сценарий использования: получил вопрос → вбил ключевое слово в поиск → нашёл карточку → переписал определение.

## Стек

Чистый HTML + CSS + JS, без фреймворков и сборщика. MathJax 3 для формул. Работает как PWA — добавляется на домашний экран телефона.

## Структура

```
index.html              точка входа
manifest.json           PWA-манифест
assets/                 иконки
src/
  css/style.css         стили
  js/
    app.js              поиск, фильтры по тегам, рендер списка
    data.js             база карточек (вопрос → определение)
    graphs.js           рендер интерактивных графиков
```

## Карточки

Каждая карточка: `{ id, tag, question, answer }`.

- `tag` — раздел курса (фильтрация по тегу вверху экрана)
- `question` — название определения/теоремы (по нему ищет поиск)
- `answer` — текст с LaTeX-формулами

## Запуск локально

```bash
python3 -m http.server 8000
# открыть http://localhost:8000
```
